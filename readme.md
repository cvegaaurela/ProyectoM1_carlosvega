# Generador de Paleta de Colores 🎨

Aplicación web interactiva para generar paletas de colores aleatorias con una interfaz moderna, elegante y responsive.

El proyecto fue desarrollado utilizando **HTML5, CSS3 y JavaScript Vanilla**, enfocado en manipulación del DOM, render dinámico, experiencia de usuario y persistencia de datos mediante `localStorage`.

# ✨ Características Principales

- Generación aleatoria de paletas de colores
- Selección dinámica de:
  - 6 colores
  - 8 colores
  - 9 colores
- Formatos de color:
  - RGBA
  - HSL
- Conversión automática a HEX
- Copiar códigos HEX al portapapeles
- Bloqueo individual de colores 🔒

# ⚙️ Funcionalidades

## 🎨 Generación dinámica de colores

La aplicación genera colores aleatorios en tiempo real según el formato seleccionado por el usuario.

---

## 🔒 Bloqueo de colores

Cada tarjeta permite bloquear colores específicos para mantenerlos mientras se generan nuevas combinaciones.

---

## 📋 Copiar código HEX

Al hacer click sobre el código HEX:

- el color se copia automáticamente al portapapeles
- aparece un toast visual indicando la acción

---

## 🧹 Limpiar paleta

La aplicación incluye un botón para:

- eliminar todas las tarjetas generadas
- limpiar el `localStorage`
- reiniciar completamente la interfaz

---

## 💾 Persistencia de datos

La aplicación almacena información usando:

```javascript
localStorage
```

Esto permite conservar:

- colores generados
- colores bloqueados
- estado actual de la paleta

incluso después de recargar la página.
